## AI 계정 운영

AI 서비스를 교육이나 업무에 쓰기 위해 계정, 권한, 결제, 보안, 개인정보 안내를 관리하는 일.

## 워크스페이스

여러 사용자가 함께 쓰는 작업 공간. ChatGPT Business, Claude Team 같은 서비스에서 자주 쓴다.

## 좌석

한 사람이 유료 기능을 쓸 수 있는 사용권. 영어로 seat라고 한다.

## 라이선스

서비스를 사용할 수 있는 권리. 좌석과 비슷하게 쓰이지만, Google Workspace처럼 앱 사용권을 뜻할 때도 많다.

## Cloud Identity Free

Google 조직 계정과 인증을 관리하기 위한 무료 라이선스. Gmail 같은 Workspace 앱 사용권과는 다르다.

## Business Starter

Google Workspace의 유료 플랜 중 하나. Gmail, Calendar 등 Workspace 앱 사용을 위해 필요한 경우가 있다.

## Colab Pro / Pro+

Google Colab의 유료 기능. 더 많은 컴퓨팅 자원과 기능을 제공한다. 조직에서는 서비스 ON과 라이선스 할당을 함께 봐야 한다.

## ChatGPT Business

조직용 ChatGPT 워크스페이스. 예전 ChatGPT Team 이름으로 부르는 경우도 있다.

## OpenAI API(Application Programming Interface) Platform

프로그램에서 OpenAI 모델을 호출하기 위한 플랫폼. ChatGPT Business와 별도다.

## API(Application Programming Interface) 키

프로그램이 API를 사용할 때 쓰는 비밀 문자열. 비밀번호처럼 다뤄야 한다.

## 서비스 계정

사람이 아니라 시스템이 API를 호출하기 위해 쓰는 계정.

## API 작업 단위

OpenAI API Platform에서 사용량, 키, 멤버, 예산을 나누는 단위.

## Cursor

AI 코드 편집기. 코드 자동완성, 설명, 수정, 디버깅을 돕는다.

## Unpaid Admin

Cursor 팀에서 비과금 관리자를 뜻한다. 팀원 관리, 결제 관리, SSO(Single Sign-On) 설정 등을 할 수 있지만 Cursor 사용 좌석은 차지하지 않는 관리자 역할이다.

## Billable Seat

과금 대상 좌석. Cursor를 실제로 사용할 수 있는 사용자로 계산되어 요금이 부과될 수 있는 좌석을 뜻한다.

## Display Bug

표시 오류. 내부 권한이나 과금 상태는 그대로인데 화면에 잘못된 이름이나 상태로 보이는 문제를 뜻한다.

## Role Dropdown

사용자 역할을 바꾸는 역할 선택 메뉴. Cursor 팀 대시보드에서는 `Member`, `Admin`, `Unpaid Admin` 같은 역할을 선택하는 메뉴를 뜻한다.

## Upcoming Invoice

다음 청구 예정 내역. 실제 과금 변경 여부를 확인할 때 Billing seat count와 함께 확인한다.

## Privacy Mode

Cursor에서 코드 데이터의 저장과 학습 사용을 제한하는 설정. 전송 자체가 없다는 뜻은 아니다.

## Claude Business / Team

Anthropic의 조직용 Claude 서비스. Primary Owner가 조직 데이터를 관리할 수 있다.

## Primary Owner

Claude 조직에서 가장 강한 관리 권한을 가진 역할. 데이터 export 권한과 관련해 수강생 안내가 필요하다.

## DPA(Data Processing Addendum)

Data Processing Addendum. 개인정보 처리와 보호 조치를 정리한 계약 부속 문서.

## SCC(Standard Contractual Clauses)

Standard Contractual Clauses. 국외 이전 등에서 쓰이는 표준계약조항.

## 국외 이전

개인정보가 해외에 있는 사업자나 서버로 제공, 보관, 처리되는 것.

## 리텐션

데이터를 얼마나 오래 보관하는지에 대한 정책.

## 디프로비저닝

사용자에게 준 계정, 좌석, 권한을 회수하는 일.

## SSO(Single Sign-On)

Single Sign-On. 조직 계정으로 여러 서비스를 한 번에 로그인하게 하는 방식.

## SCIM(System for Cross-domain Identity Management)

System for Cross-domain Identity Management. 사용자 생성, 수정, 삭제를 자동화하는 계정 관리 표준. 장기 운영에서는 매우 유용하다.

## OU(Organizational Unit)

Organizational Unit. Google Workspace에서 사용자를 묶는 조직 단위.

## DLP(Data Loss Prevention)

Data Loss Prevention. 민감정보가 외부로 나가지 않도록 탐지하거나 차단하는 기능.

## RBAC(Role-Based Access Control)

Role-Based Access Control. 역할에 따라 접근 권한을 나누어 주는 권한 관리 방식.

## SAML(Security Assertion Markup Language)

Security Assertion Markup Language. 조직 계정으로 외부 서비스에 로그인할 때 쓰는 인증 연동 표준.

## OIDC(OpenID Connect)

OpenID Connect. OAuth 2.0 위에서 사용자 인증 정보를 확인하는 인증 연동 표준.

## OTP(One-Time Password)

One-Time Password. 한 번만 사용할 수 있는 인증번호 또는 인증 코드.

## ACH(Automated Clearing House)

Automated Clearing House. 미국 은행 계좌 기반 이체망. 해외 서비스 결제 안내에서 카드 외 결제수단 예시로 등장할 수 있다.

## PO(Purchase Order)

Purchase Order. 구매 주문서. 회계·구매 절차에서 사전 승인된 주문 문서로 쓰인다.

## CSV(Comma-Separated Values)

Comma-Separated Values. 쉼표로 항목을 구분하는 표 형식 텍스트 파일.

## SaaS(Software as a Service)

Software as a Service. 설치형 소프트웨어가 아니라 웹 서비스 형태로 구독해 쓰는 소프트웨어.

## LLM(Large Language Model)

Large Language Model. 대량의 텍스트를 학습해 문장을 이해하고 생성하는 대규모 언어 모델.

## True-up

연간계약 중 실제 좌석 증가분을 주기적으로 정산하는 방식.

## 선결제

앞으로 쓸 비용 일부를 미리 내는 것. Google Workspace에서는 신규 계정 제한을 풀기 위한 수동 결제와 연결될 수 있다.
