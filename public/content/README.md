# AI 계정 운영 위키 묶음

이 폴더는 AI 교육 계정 운영을 위한 Markdown 위키 묶음이다.

## 구성

- `Home.md`: 위키 첫 화면
- `_Sidebar.md`: GitHub Wiki용 사이드바
- `services/`: 서비스별 운영 문서
- `billing/`: 결제·청구·좌석 문서
- `compliance/`: 개인정보·보안·삭제·사고 대응 문서
- `education/`: 교육 담당자와 교육생 문서
- `templates/`: 바로 고쳐 쓸 수 있는 안내문·동의서
- `qna/`: 프로젝트 실제 질의응답 반영 문서
- `glossary.md`: 쉬운 용어 사전
- `SOURCES.md`: 공식 출처 모음

## 사용 방법

GitHub Wiki에 넣을 때는 `Home.md`, `_Sidebar.md`를 루트에 두면 된다.

Confluence에 넣을 때는 `Home.md`를 최상위 페이지로 만들고, 각 폴더를 페이지 묶음처럼 만들면 된다. Markdown을 직접 가져오기 어렵다면 본문을 복사해서 붙여 넣으면 된다.

## 주의

이 문서는 운영 가이드다. 법률 자문은 아니다. 실제 동의서, 국외이전 고지, 계약 문구는 법무·개인정보보호책임자·정보보안 담당자의 검토를 거쳐야 한다.
