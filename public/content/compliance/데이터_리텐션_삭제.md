## 개요

AI 서비스에서 “사용자를 제거했다”와 “데이터를 삭제했다”는 같은 말이 아니다.

이 문서는 교육 종료 후 어떤 데이터가 남을 수 있는지, 무엇을 삭제할 수 있는지, 무엇을 사용자가 직접 해야 하는지 정리한다.

## 리텐션이란

리텐션(retention)은 데이터를 얼마 동안 보관하는지를 뜻한다.

쉬운 말로는 “언제까지 남아 있나”다.

## ChatGPT Business / OpenAI

### 대화 삭제

OpenAI 도움말은 채팅을 삭제하면 계정 화면에서 즉시 제거되고, OpenAI 시스템에서 30일 내 영구 삭제 예정이라고 설명한다. 다만 이미 비식별화되었거나, 보안 또는 법적 의무로 더 오래 보관해야 하는 경우는 예외다.

### 계정 삭제

OpenAI 계정 삭제는 영구적이고 되돌릴 수 없다. 계정 삭제 시 데이터는 30일 내 삭제된다고 안내된다. 단, 법에서 요구하거나 허용하는 일부 데이터는 더 오래 보관될 수 있다.

### 운영상 주의

- 워크스페이스에서 멤버를 제거해도 계정 자체가 삭제되는 것은 아니다.
- 교육생 개인 계정의 삭제는 사용자가 직접 해야 할 수 있다.
- 조직은 “서비스 접근 회수”와 “데이터 삭제 요청”을 구분해서 안내해야 한다.

## Cursor

Cursor 보안 문서는 사용자가 Settings dashboard의 Advanced → Delete Account에서 계정을 삭제할 수 있으며, 계정 관련 모든 데이터와 indexed codebases를 포함해 30일 내 완전 제거를 보장한다고 설명한다.

Privacy Mode가 켜진 경우에는 코드 데이터가 모델 제공자에 저장되거나 학습에 사용되지 않는다고 설명한다. 팀 멤버는 기본적으로 Privacy Mode가 강제된다.

### 운영상 주의

- 팀에서 멤버를 제거하는 것과 사용자가 계정을 삭제하는 것은 다르다.
- 코드베이스 인덱싱 데이터가 있을 수 있으므로 계정 삭제 또는 인덱싱 설정을 별도로 봐야 한다.
- `.cursorignore`와 실습용 레포 사용이 중요하다.

## Claude Business / Team

Anthropic 문서에 따르면 조직에서 사용자가 제거되면 남은 멤버는 그 사용자의 chats에 접근할 수 없다. 하지만 제거된 사용자의 데이터는 조직 Primary Owner의 data export에 포함될 수 있다고 안내된다.

또한 Anthropic의 조직 데이터 보존 문서는 API 입력·출력이 기본적으로 30일 내 백엔드에서 삭제된다고 설명하지만, 법적 준수, 사용 정책 집행, 별도 합의 등 예외가 있다.

Enterprise에는 custom retention controls가 있으나, 관련 문서 기준 최소 retention은 30일이다.

## Google Workspace / Colab

Google Workspace에서는 사용자 계정, 라이선스, 서비스 데이터가 분리된다.

- 라이선스 회수: 유료 기능 사용권 제거
- 계정 정지: 로그인 제한
- 계정 삭제: 사용자 계정 삭제
- 데이터 이전/보관: Drive, Gmail 등 데이터 처리
- Colab 라이선스 회수: Colab Pro/Pro+ 기능 제거

Colab 교육 후에는 라이선스 회수와 서비스 접근 권한을 확인한다. 노트북 파일이 Drive에 저장되어 있다면 Drive 데이터 정책도 함께 봐야 한다.

## 삭제 요청 안내 예시

```text
교육 종료 후 서비스 접근 권한은 운영자가 회수합니다.
다만 사용자의 개인 계정 자체 삭제나 개인 대화 삭제는 서비스별 정책에 따라 사용자가 직접 진행해야 할 수 있습니다.
삭제를 원하는 경우 교육 담당자에게 문의해 주시면 서비스별 절차를 안내드립니다.
```

## 자주 헷갈리는 점

### 멤버 제거와 삭제는 왜 다른가?

멤버 제거는 접근권을 끊는 것이다. 삭제는 데이터 자체를 없애는 것이다. 서비스는 보통 이 둘을 따로 처리한다.

### 30일이면 무조건 모두 삭제되는가?

아니다. 보안, 법적 의무, 비식별화, 사용 정책 집행 같은 예외가 있을 수 있다.

### 교육생에게 “다 삭제됩니다”라고 안내해도 되나?

권장하지 않는다. “서비스별 정책에 따라 삭제 또는 보관될 수 있다”고 안내한다.
