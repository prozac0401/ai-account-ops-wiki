## 개요

AI 교육은 보안이 빡빡한 환경이 아니어도 사전 검토가 필요하다.

검토의 목적은 교육을 막는 것이 아니다. 어디까지 허용하고, 무엇은 금지하고, 사고가 나면 누구에게 말할지를 미리 정하는 것이다.

## 본 프로젝트의 실제 전제

기존 질의응답에서 정리된 전제는 다음과 같다.

- 수강생이 개인 Gmail 계정을 사용할 수 있다.
- Google Form으로 이메일 주소를 수집할 수 있다.
- 수집된 이메일을 ChatGPT Business와 Cursor 계정에 등록한다.
- 교육장은 보안 규제가 비교적 자유로운 편이다.
- 그래도 정보보안팀 문의가 필요하다.

이 경우 거창한 보안 심사보다 “최소 통제”를 명확히 하는 방식이 현실적이다.

## 4대 핵심 검토 항목

### 1. 계정 관리

확인할 것:

- Gmail 계정 초대 방식이 허용되는가.
- 교육 종료 후 회수 절차가 있는가.
- 2단계 인증을 권장 또는 의무로 할 것인가.
- 계정 공유를 금지할 것인가.
- 초대 수락 현황을 누가 확인할 것인가.

### 2. 데이터 입력·저장

확인할 것:

- 실제 업무 데이터 입력을 금지할 것인가.
- 개인정보, 고객정보, 회사기밀, API 키 입력 금지 문구가 있는가.
- 실습용 샘플 데이터를 제공하는가.
- 수강생이 파일 업로드를 해도 되는가.
- 로그를 얼마나 보관할 것인가.

### 3. 모델 서버 전송 데이터

확인할 것:

- ChatGPT와 Cursor의 데이터 처리 정책을 확인했는가.
- 모델 학습 제외 또는 Privacy Mode 관련 공식 문서를 확인했는가.
- 국외 이전 고지 대상인지 검토했는가.
- DPA 또는 보안 인증 자료를 보관할 것인가.

### 4. 사고 대응

확인할 것:

- 사고 의심 시 누구에게 연락하는가.
- 계정을 누가 즉시 비활성화하는가.
- API 키 노출 시 누가 키를 폐기하는가.
- 프롬프트나 파일 삭제 요청은 어떻게 하는가.
- 보고 시간 기준은 무엇인가.

## 정보보안팀 문의 메일 예시

```text
제목: AI 교육용 SaaS(ChatGPT Business·Cursor) 임시 계정 운영 관련 최소 보안 검토 요청

안녕하세요.

[교육명] 과정에서 수강생이 생성형 AI 서비스를 실습용으로 사용할 예정입니다.
사용 서비스는 ChatGPT Business와 Cursor이며, 수강생은 개인 Gmail 주소로 초대될 수 있습니다.
이메일 주소는 Google Form으로 수집하고, 교육 종료 후 서비스 접근 권한은 회수할 예정입니다.

교육 환경은 일반 업무망보다 보안 규제가 비교적 완화된 교육장이나,
개인정보와 회사 중요정보 입력을 막기 위한 최소 통제 절차는 사전에 확인하고자 합니다.

아래 사항에 대해 검토 의견 부탁드립니다.

1. Gmail 계정 초대 방식과 교육 종료 후 권한 회수 절차의 적정성
2. 실제 업무정보·개인정보·비밀키 입력 금지 안내로 충분한지 여부
3. ChatGPT/Cursor의 데이터 처리 정책 및 모델 학습 제외/Privacy Mode 문서 확인 필요 여부
4. 사고 의심 시 계정 차단, API 키 폐기, 보고 라인의 적정성

첨부 예정:
- 수강생 안내문
- 개인정보 고지서·동의서 초안
- 서비스별 공식 보안/개인정보 문서 링크
- 계정 회수 체크리스트

감사합니다.
```

## 교육장 완화 환경에서의 최소 통제

| 통제 | 최소 기준 |
|---|---|
| 계정 | 개인 계정 공유 금지, 초대 명단 보관 |
| 데이터 | 실습용 샘플 데이터만 사용 |
| 인증 | 2단계 인증 권장, 조직 계정이면 SSO 우선 |
| 로그 | 초대/회수/결제/사고 기록 보관 |
| 사고 | 1차 연락 담당자와 차단 담당자 지정 |
| 종료 | 교육 종료 당일 또는 익일 권한 회수 |

## 자주 헷갈리는 점

### 보안 규제가 완화된 교육장이면 검토가 없어도 되나?

아니다. 완화된 환경이라는 말은 통제를 낮출 수 있다는 뜻이지, 책임이 사라진다는 뜻은 아니다.

### 수강생이 개인 Gmail을 쓰면 회사 보안과 무관한가?

아니다. 회사가 교육을 위해 외부 SaaS 계정에 등록하고 교육 과정에서 데이터를 입력하게 한다면 운영 책임이 생긴다.

### DLP를 못 쓰면 교육을 못 하나?

아니다. 짧은 교육이라면 안내, 샘플 데이터, 금지 항목, 사고 대응 라인을 최소 통제로 쓸 수 있다.
