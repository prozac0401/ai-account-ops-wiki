# 개인정보 고지와 동의

[목차]

## 개요

AI 교육 계정 운영에서는 이메일 주소 하나를 받더라도 개인정보 처리가 시작된다.

여기에 ChatGPT, Cursor, Claude 같은 해외 SaaS(Software as a Service)<sup class="wiki-footnote-ref"><a class="wiki-footnote-link" id="abbr-ref-2" href="#abbr-note-2">[2]</a></sup>를 붙이면 프롬프트, 코드, 첨부파일, 로그가 해외로 전송될 수 있다. 그래서 단순한 서비스 안내문만으로는 부족할 수 있다.

## 기본 원칙

수강생에게는 두 겹으로 안내한다.

1. **내부 고지**: 한국어로 목적, 항목, 보유기간, 거부권, 국외 이전을 쉽게 설명한다.
2. **외부 정책 링크**: OpenAI, Cursor, Google, Anthropic의 공식 개인정보·보안 문서를 링크한다.

영어 원문만 던지는 방식은 실무상 좋지 않다. 수강생은 무엇에 동의하는지 알아야 한다.

## 개인정보보호법상 기본 고지 항목

개인정보 수집·이용 동의를 받을 때는 다음 항목이 핵심이다.

| 항목 | 쉬운 말 |
|---|---|
| 수집·이용 목적 | 왜 이메일과 이름을 받는가 |
| 수집 항목 | 무엇을 받는가 |
| 보유·이용 기간 | 언제까지 갖고 있는가 |
| 동의 거부권과 불이익 | 거부할 수 있는가, 거부하면 무엇을 못 하는가 |

## 국외 이전 고지

개인정보보호위원회의 국외이전 제도 안내는 개인정보 보호법 제28조의8에 따른 국외 이전 요건으로 정보주체 별도 동의, 법률·조약 근거, 계약 체결·이행에 필요한 처리위탁/보관과 고지, 인증 또는 동등 보호 수준 인정 등을 제시한다.

AI SaaS(Software as a Service) 교육에서는 보통 다음 정보가 필요하다.

- 이전받는 자
- 이전 국가
- 이전되는 항목
- 이전 목적
- 보유·이용 기간
- 이전 방법
- 보호 조치
- 동의 거부 시 불이익

## 서비스별 고지 포인트

| 서비스 | 고지할 내용 |
|---|---|
| ChatGPT Business | 이메일, 이름, 프롬프트, 답변, 첨부파일, 로그가 처리될 수 있음. business data는 기본적으로 모델 학습 제외로 안내됨 |
| OpenAI API | API 입력·출력, 로그, 키 사용 정보가 처리될 수 있음. API 작업 단위와 키 관리 필요 |
| Cursor | 코드, 주석, 프롬프트, IDE(Integrated Development Environment)<sup class="wiki-footnote-ref"><a class="wiki-footnote-link" id="abbr-ref-1" href="#abbr-note-1">[1]</a></sup> 이벤트가 처리될 수 있음. Privacy Mode가 있어도 전송 자체는 발생 가능 |
| Google Workspace/Colab | 계정, 이메일, 라이선스, Colab 사용 정보가 Google 조직 계정에서 처리됨 |
| Claude Business | 대화, 파일, 사용 패턴이 조직 Primary Owner의 관리 대상이 될 수 있음 |

## 동의서와 고지서의 차이

| 문서 | 목적 | 서명 필요 여부 |
|---|---|---|
| 고지서 | 내용을 알려 주기 | 보통 확인 기록을 남김 |
| 동의서 | 사용자가 동의했음을 남기기 | 체크박스 또는 서명 필요 |

현장에서는 두 문서를 분리하는 방식이 논의되었다. 실무적으로는 “고지서 1부 + 동의서 1부”가 관리하기 쉽다.

## 동의 거부 문구 예시

```text
귀하는 개인정보 수집·이용 및 국외 이전에 대한 동의를 거부할 수 있습니다.
다만 동의하지 않을 경우, 본 교육에서 제공되는 생성형 AI 서비스 계정 및 관련 기능을 이용할 수 없습니다.
대체 과제 또는 AI 미사용 실습 제공 여부는 교육 과정별로 별도 안내됩니다.
```

## 수강생에게 꼭 말할 것

동의서에는 법적 문구가 들어간다. 하지만 수강생이 실제로 기억해야 할 문장은 더 짧아야 한다.

- 이 서비스는 해외 AI 서비스다.
- 입력한 내용이 서비스 제공자 서버로 전송될 수 있다.
- 교육용 샘플 데이터만 써야 한다.
- 개인정보, 고객정보, 회사기밀, API 키를 넣으면 안 된다.
- 교육이 끝나면 권한은 회수된다.

## 자주 헷갈리는 점

### Privacy Mode가 있으면 국외 이전 동의가 필요 없나?

그렇게 단정하기 어렵다. Privacy Mode는 보관과 학습 사용을 제한하는 장치에 가깝다. AI 기능을 쓰기 위해 해외 서버로 전송되는 구조라면 국외 이전 고지를 검토해야 한다.

### 수강생 이메일만 받는데 동의서가 필요한가?

이메일 주소도 개인정보다. 특히 이 이메일을 외부 SaaS에 등록한다면 수집·이용 목적과 제공/이전 구조를 설명해야 한다.

### OpenAI, Cursor의 원문 정책 링크만 주면 충분한가?

권장하지 않는다. 내부 한국어 요약 고지와 원문 링크를 함께 제공하는 방식이 이해도와 증빙 측면에서 낫다.
