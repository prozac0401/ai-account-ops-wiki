## 안내문 예시

```text
[AI 실습 계정 사용 안내]

안녕하세요.
이번 교육에서는 ChatGPT Business, Cursor, Google Colab 등 생성형 AI 서비스를 실습에 사용합니다.

아래 내용을 확인한 뒤 교육에 참여해 주세요.

1. 로그인 계정
- 교육 신청 시 제출한 이메일 주소로 로그인해 주세요.
- 다른 이메일로 로그인하면 유료 기능이나 팀 워크스페이스가 보이지 않을 수 있습니다.

2. 초대 메일
- 교육 전까지 각 서비스의 초대 메일이 발송될 수 있습니다.
- 초대 메일을 받은 경우, 교육 전 미리 수락해 주세요.

3. 입력 금지 정보
아래 정보는 AI 서비스에 입력하거나 업로드하지 않습니다.
- 개인정보: 주민등록번호, 연락처, 주소 등
- 고객정보: 고객명, 상담 이력, 계약 정보 등
- 회사 기밀: 내부 문서, 제안서, 견적서, 비공개 소스코드 등
- 인증정보: API 키, 비밀번호, 토큰, 인증서 등

4. 실습 데이터
- 교육 중에는 강사가 제공하는 샘플 데이터만 사용합니다.
- 개인 업무 자료나 회사 자료를 임의로 사용하지 않습니다.

5. 계정 회수
- 교육 종료 후 교육용 서비스 접근 권한은 회수됩니다.
- 개인 계정 자체 삭제나 대화 삭제는 서비스별 정책에 따라 별도 절차가 필요할 수 있습니다.

6. 문제 발생 시
- 로그인 문제, 초대 메일 미수신, 유료 기능 미노출, 민감정보 입력 실수는 즉시 교육 담당자에게 알려 주세요.

감사합니다.
```

## 짧은 버전

```text
AI 실습에서는 교육용 샘플 데이터만 사용합니다.
개인정보, 고객정보, 회사 기밀, API 키, 비밀번호는 ChatGPT·Cursor·Colab 등 어떤 AI 서비스에도 입력하지 마세요.
계정은 본인에게 안내된 이메일로만 사용하며, 교육 종료 후 접근 권한은 회수됩니다.
```

## 현장 구두 안내 버전

```text
오늘 실습에서는 샘플 데이터만 씁니다.
실제 고객정보, 회사 문서, API 키, 비밀번호는 넣지 마세요.
실수로 넣었다면 바로 알려 주세요. 빨리 알려 주는 것이 가장 안전합니다.
```
